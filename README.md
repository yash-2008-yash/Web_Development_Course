# Web Development Course

## Overview  
This repository shows my complete web-development learning journey. It includes every chapter — starting from HTML basics, through styling with CSS, to JavaScript fundamentals and beyond — giving a structured, progressive approach to learning web development.

---

## What can you find inside the repository?

| Topic | Description |
|-------------------|-------------|
| **HTML Fundamentals** | Structure of web pages: basic structure, headings, paragraphs, links, images, lists, tables — up to semantic elements and media. |
| **CSS Styling** | Styling webpages: inline/internal/external CSS, selectors, box model (margin, padding, border), font/text/color properties, layout (flexbox & grid), animations, responsive design with media queries, and more. |
| **JavaScript Basics** | Core JS topics: variables, data types, operators, conditionals, loops, functions, DOM manipulation, events, asynchronous programming (Promises / async-await), error handling, and object-oriented JS. |
| **Hands-on Exercises & Projects** | Practical exercises and small projects to reinforce concepts, e.g. media players, dynamic websites, a “Netflix-clone” project, interactive JS exercises (e.g. business-name generator, calculator), and real-world style practice. |
